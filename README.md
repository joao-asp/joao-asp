### Projetos & Atuação

* **Assaltar as Nuvens (Pesquisa PIBIC - UNICAMP):** Investigação acadêmica sociotécnica voltada ao estudo de redes comunitárias, tecnodiversidade e apropriação tecnológica popular, focando na infraestrutura autônoma de comunicação.
* **Cartografia Climática Periférica:** Projeto de [Geração Cidadã de Dados (GCD)](https://datalabe.org/geracao-cidada-de-dados/) atualmente em execução com apoio do edital [JUCAMP](https://www.instagram.com/toemmovimento/). Atuo na concepção e deploy da infraestrutura física (Kit-Soberania com Raspberry Pi e bancos de dados locais) para dar suporte a mapeamentos participativos e painéis de denúncia contra o racismo ambiental no bairro Campo Belo.
* **[grana.noiz](https://github.com/joao-asp/grana.noiz):** Sistema open-source via Telegram para gestão financeira de organizações. Automatiza cobranças, extratos e aumenta a transparência do caixa de coletivos. Construído com Python e deploy containerizado.
* **Apple Developer Academy:** Formação imersiva focada no ciclo completo de desenvolvimento de software. Experiência prática com prospecção de produtos, design de interfaces (UX/UI) e metodologias ágeis para a construção de aplicativos centrados no usuário.
* **Youth Brasil 2026 (CGI.br / NIC.br):** Participante do programa de formação em Governança da Internet, com foco nos fundamentos técnicos, arquitetura de redes e implicações sociotécnicas da infraestrutura digital.
